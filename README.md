# Worddee.ai

Worddee.ai คือเว็บแอปพลิเคชันสำหรับฝึกแต่งประโยคภาษาอังกฤษ โดยมีระบบ AI ช่วยตรวจสอบความถูกต้องและให้คำแนะนำ ฟีเจอร์หลักประกอบด้วย "Word of the Day" (คำศัพท์ประจำวัน) และแดชบอร์ดสำหรับติดตามพัฒนาการของผู้เรียน

## ฟีเจอร์หลัก

-   **Word of the Day (คำศัพท์ประจำวัน)**: เรียนรู้คำศัพท์ใหม่ๆ พร้อมความหมาย และฝึกแต่งประโยคจากคำศัพท์นั้น
-   **AI Feedback (ระบบตรวจสอบด้วย AI)**: ตรวจสอบความถูกต้องของไวยากรณ์และความเกี่ยวข้องของประโยคทันที พร้อมให้คะแนนโดย AI (ผ่าน n8n)
-   **Dashboard (แดชบอร์ด)**: ติดตามความสม่ำเสมอในการเรียน (Streaks) และดูพัฒนาการผ่านกราฟและสถิติต่างๆ
-   **Dockerized**: รันโปรเจกต์ได้ง่ายๆ ด้วย Docker Container

## เทคโนโลยีที่ใช้

-   **Frontend**: Next.js (React), Tailwind CSS, Recharts
-   **Backend**: FastAPI (Python)
-   **AI/Workflow**: n8n
-   **Database**: (เชื่อมต่อผ่าน n8n หรือใช้ข้อมูลจำลองสำหรับโปรเจกต์นี้)

## การติดตั้งและใช้งาน

1.  **สิ่งที่ต้องมี**: ตรวจสอบให้แน่ใจว่าเครื่องติดตั้ง Docker และ Docker Compose แล้ว
2.  **ดาวน์โหลดโปรเจกต์**: Clone หรือดาวน์โหลดไฟล์โปรเจกต์นี้
3.  **ตั้งค่า Environment**: ตรวจสอบไฟล์ `.env` (มีการตั้งค่าเริ่มต้นไว้ให้แล้ว)
4.  **รันโปรแกรม**: เปิด Terminal แล้วพิมพ์คำสั่ง:

    ```bash
    docker-compose up --build
    ```

5.  **เข้าใช้งาน**:
    -   **Frontend (หน้าเว็บ)**: [http://localhost:3000](http://localhost:3000)
    -   **Backend API**: [http://localhost:8000](http://localhost:8000)
    -   **n8n Workflow**: [http://localhost:5678](http://localhost:5678)

## โครงสร้างโปรเจกต์

-   `frontend/`: ซอร์สโค้ดของหน้าเว็บ (Next.js)
-   `backend/`: ส่วนของเซิร์ฟเวอร์และ API (FastAPI)
-   `n8n/`: ข้อมูลและการตั้งค่า Workflow ของ n8n
-   `docker-compose.yml`: ไฟล์สำหรับจัดการ Service ทั้งหมดใน Docker

## API Endpoints

-   `GET /api/word`: สุ่มคำศัพท์
-   `POST /api/validate-sentence`: ตรวจสอบประโยคที่ผู้ใช้ส่งมา
-   `GET /api/summary`: ดึงข้อมูลสรุปผลการเรียนสำหรับหน้า Dashboard
